#XPRefresh
Swift 版下拉刷新控件

#Use:
```
tableView.xp_header = Header.init {
	print("正在刷新")
}

tableView.xp_footer = Footer.init{
	print("上拉正在刷新")
	self.numbers+=5
}
```
